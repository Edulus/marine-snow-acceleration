# In Situ Sargassum Ballasting

## Purpose

This document defines the in situ sargassum ballasting pathway.

## Status

Draft placeholder.

## Core Concept

Apply a marine-safe ballast formulation directly to floating pelagic sargassum over sufficiently deep water. The formulation should cling to the biomass, increase density, collapse buoyancy, and cause the sargassum to sink before coastal landfall.

## Core Thesis

> Apply ballast directly to offshore sargassum in sufficiently deep water, causing nuisance biomass to sink before landfall and converting a coastal-management problem into a potential carbon-export pathway.

## Key Requirements

- Treat floating mats in place
- Avoid collection-and-transport dependency
- Deploy only over sufficient depth
- Target sargassum likely to cause coastal harm
- Verify sinking
- Estimate storage duration
- Count lifecycle emissions
- Monitor chemical and ecological effects

## Candidate Ballast Families

- Clay-mineral slurry
- Biodegradable binder-mineral slurry
- Two-part marine-snow glue system
- Calcium carbonate or limestone coating
- Basalt or silicate mineral coating
- Iron-rich mineral coating
- Other tested non-toxic dense coatings
