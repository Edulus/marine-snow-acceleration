# Monitoring, Reporting, and Verification Architecture

## Purpose

This document defines how Marine Snow Acceleration interventions can be measured and verified.

## Status

Draft placeholder.

## MRV Doctrine

A project that cannot be measured cannot be credited.

A project that cannot distinguish its effects from natural variability cannot claim removal.

A project that cannot quantify uncertainty cannot claim precision.

## Measurement Targets

- baseline conditions,
- treated area,
- biomass treated,
- ballast applied,
- fraction that sinks,
- sinking speed,
- depth reached,
- carbon retained,
- decomposition products,
- air-sea gas exchange,
- oxygen impact,
- benthic footprint,
- avoided coastal landfall,
- avoided beach decomposition,
- lifecycle emissions.

## Possible Tools

- satellite observations,
- aircraft mapping,
- drones,
- vessels,
- drifting floats,
- particle traps,
- chemical tracers,
- oxygen sensors,
- carbon chemistry sampling,
- benthic monitoring systems.
