# Pilot Design

## Purpose

This document will define the minimum responsible pilot experiment.

## Status

Draft placeholder.

## Pilot Principles

- Start small
- Use controlled conditions first
- Measure baseline conditions
- Use comparison/control areas
- Define shutdown triggers
- Monitor before, during, and after treatment
- Avoid carbon-crediting claims before verification
- Publish results, including failures

## Candidate Pilot Sequence

1. Laboratory adhesion and sinking tests
2. Tank-scale decomposition and gas-retention tests
3. Mesocosm ecological tests
4. Small bounded field trial, if permitted
5. Expanded pilot only after independent review
